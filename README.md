# keyword-cipher-tool
A simple HTML tool that encrypts and decrypts messages using a keyword-based substitution cipher. Works on all devices.


## Features

- Encrypt text using any keyword
- Decrypt text with the same keyword
- Works on all devices (mobile & desktop)

## How to Use

1. Open `enigma.html` in any browser
2. Enter your text and a keyword
3. Click Encrypt or Decrypt

## How It Works

1. Enter a keyword (e.g., "SECRET")
2. The alphabet is rearranged using the keyword
3. Each letter is substituted to create cipher text
4. To decrypt, use the same keyword

## Example

- **Keyword:** SECRT (duplicates removed)
- **Input:** HELLO
- **Output:** FADDP

## Tech

Built entirely in a single HTML file with embedded CSS and JavaScript.
